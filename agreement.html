Пользовательское соглашение

1. Общие положения

1.1. Настоящее Пользовательское соглашение (далее — Соглашение) регулирует отношения между ТОО Toka Dev (далее — Компания) и любым лицом (далее — Пользователь), использующим мобильное приложение CMR (далее — Приложение).

1.2. Используя Приложение, Пользователь подтверждает, что ознакомлен с условиями настоящего Соглашения и принимает их в полном объеме.

1.3. Компания оставляет за собой право вносить изменения в настоящее Соглашение. Продолжение использования Приложения после внесения изменений означает принятие Пользователем новых условий.

2. Предмет соглашения

2.1. Компания предоставляет Пользователю доступ к Приложению, которое позволяет:

  • Создавать и управлять заказами на строительные материалы;
  • Взаимодействовать с поставщиками строительных материалов;
  • Получать информацию о ценах и наличии товаров;
  • Использовать другие функции, доступные в Приложении.

3. Права и обязанности сторон

3.1. Пользователь имеет право:

  • Использовать Приложение в соответствии с его функциональным назначением;
  • Получать техническую поддержку;
  • Обращаться с предложениями по улучшению работы Приложения.

3.2. Пользователь обязуется:

  • Предоставлять достоверную информацию при регистрации и использовании Приложения;
  • Не нарушать работу Приложения;
  • Не использовать Приложение для незаконной деятельности;
  • Не передавать доступ к своему аккаунту третьим лицам.

4. Ответственность сторон

4.1. Компания не несет ответственности за:

  • Невозможность использования Приложения по причинам, не зависящим от Компании;
  • Потерю данных Пользователя;
  • Действия третьих лиц, совершенные с использованием аккаунта Пользователя.

4.2. Пользователь несет ответственность за:

  • Достоверность предоставленной информации;
  • Сохранность данных для входа в Приложение;
  • Действия, совершенные с использованием его аккаунта.

5. Интеллектуальная собственность

5.1. Все права на Приложение, включая его дизайн, программный код, базы данных, принадлежат Компании.

5.2. Пользователь не вправе копировать, модифицировать, распространять или использовать Приложение и его компоненты без письменного разрешения Компании.

6. Заключительные положения

6.1. Настоящее Соглашение регулируется законодательством Республики Казахстан.

6.2. Все споры, возникающие между сторонами, разрешаются путем переговоров. При невозможности достижения согласия споры разрешаются в судебном порядке.
